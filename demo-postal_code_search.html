DEMO - postal code search using jquery mobile

This is a demo to show how to make a postal code search from the French postal codes using the listview component of JqueryMobile.

This code can be adapted to make a serach inside US ZIP codes.

A small modification of jquery mobile 1.1.0 needs to be done to perform a this demo. The file that contains this modification is jquery.mobile-1.1.0_edited.js

By default the list is empty because it's content generated only. A callback function needs to be implemented to dynamicaly generate/append then content items. This callback is binded on listview.options.onkeyupcallback Javascript code must be written to handle this. (see postalcodesearch.js)

When you click on a item the result is printed on the output console of your browser (so open the console to see the result). Please note that there is still a small bug: you need to click twice on an item to choose it.
